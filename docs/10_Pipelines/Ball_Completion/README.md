# 🧠 丢球段补全（球员反应推断 + 端点融合）

## 目标 / 范围（论文级创新）
当 A/B 均未捕捉到球的一段时间内，利用球员反应（朝向/加速/聚集趋势等）推断球的似然场，
并与丢失段前后观测融合修正，输出更可信的球轨迹。

## 输入
- 丢球段前后可观测球位置（端点约束）
- 丢球段内球员轨迹（最好世界坐标；也可先像素轨迹）
-（可选）姿态/朝向/微动作线索（后续增强）

## 输出（关键工件）
- 补全后的球轨迹（completed）
- 最小评测集（20 个丢球片段）+ 对比实验记录

## 依赖（上游/下游）
- 上游：球员轨迹稳定性、标定（可选但强建议）
- 下游：事件理解、评分、论文撰写

## 当前进度 & 已知问题
- ✅ 创新点明确
- ⚠️ 需要最小数据集 + 统一评测协议来证明收益

## 指标 / 验收标准（建议先三项）
- 端点误差（与端点观测一致）
- 速度连续性（物理合理）
- 事件一致性（接球/射门等位置合理）

## 下一步（可执行）
