# 🎥 采集与机位管线（Acquisition）

## 目标 / 范围
为“球员轨迹 + 球轨迹”提供稳定视频输入，并降低丢球/遮挡风险。
机位规划：固定机位 A/B（战术视角）为主，跟球机为辅。

## 输入
- 视频流：A/B 固定机位（高度已确认 3m）、跟球机（现有素材多）
-（未来）球员定位器时序数据（球无定位器）

## 输出（关键工件）
- A/B 机位参数表（位置/高度/朝向/FOV假设）
- 覆盖示意 & 遮挡风险区说明
- 像素估算：球直径(px)随距离分段（用于判断可检出性）

## 依赖（上游/下游）
- 下游依赖：标定、检测跟踪、球检测都依赖采集质量
- 同步依赖：多机位与定位器最终要时间对齐

## 当前进度 & 已知问题
- ✅ A/B 高度=3m；目标：A/B 尽量别同时挡球（至少一个机位看得到球）
- ⚠️ 素材结构偏跟球视角：不利于长序列稳定跟踪（急需固定机位/高处广角）

## 指标 / 验收标准
- 关键区域球可见率（至少一个机位）
- 球在关键区域像素足够（可检测的最小px阈值）
- 多机位时间对齐误差（目标：帧级/亚秒级）

## 下一步（可执行）
- 产出覆盖&遮挡风险图 + 像素估算表（先用估计参数也行）
- 补齐固定机位/高处广角样本（哪怕先网上同类素材预训练）
