# 📐 标定与场地映射管线（Calibration）

## 目标 / 范围
建立像素 ↔ 世界坐标映射，让轨迹/速度/距离/战术指标在世界坐标可用、可复现。

## 输入
- 场地图像帧（含场线/角点/门框等几何线索）
- 球场几何定义（world_geometry.yml）

## 输出（关键工件）
- 相机标定结果：cam_*.json（每个机位一份）
- world_geometry yml（版本化）
- 标定 Runbook（从 0 到跑通）

## 依赖（上游/下游）
- 上游依赖：采集画面可见场线/角点
- 下游依赖：球员世界轨迹、球轨迹、论文指标都依赖标定稳定

## 当前进度 & 已知问题
- ⚠️ 曾遇到环境/包路径问题（例如 No module named 'ml'）
- ⚠️ 这是“地基”，不稳会拖慢后续一切

## 指标 / 验收标准
- 重投影误差（或关键点误差）在可接受范围
- 同一机位重复标定结果稳定（漂移小）
- 输出文件格式固定 & 可复现
