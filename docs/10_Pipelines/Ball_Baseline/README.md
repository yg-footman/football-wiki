# ⚽ 球轨迹 baseline（A/B 融合）

## 目标 / 范围
在“球无定位器”的约束下，用 A/B 视觉观测构建球的连续轨迹 baseline（先 2D）。

## 输入
- A/B 视频观测到的球候选位置
- 标定映射（像素→世界）
-（可选）球员世界轨迹作为先验约束

## 输出（关键工件）
- 球轨迹 baseline（整场）
- 基础平滑/物理约束后的轨迹

## 依赖（上游/下游）
- 上游：Calibration、Vision 检测质量
- 下游：丢球段补全、事件/动作理解

## 当前进度 & 已知问题
- ✅ 事实约束明确：球只能靠视觉融合
- ⚠️ 丢球段不可避免，需要补全机制

## 指标 / 验收标准
- 连续性（缺失段长度、断点数）
- 速度/加速度合理（物理可解释）
- 与事件点一致性（接球/射门位置抽检）

## 下一步（可执行）
- 先用“人工标球点”的短片跑通端到端 baseline（最小验证）
