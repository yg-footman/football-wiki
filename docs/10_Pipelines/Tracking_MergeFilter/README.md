# 🔗 轨迹合并与过滤管线（Tracklets → Clean Tracks）

## 目标 / 范围
将碎片化 tracklets 合并为长轨迹，并过滤掉明显错误/短/漂轨迹，输出稳定球员轨迹。

## 输入
- tracklets（原始跟踪输出）
- 合并/过滤参数（时序间隙、运动一致性、置信度等）

## 输出（关键工件）
- merged_tracks
- filtered_tracks
- 统计与可视化：数量、时长分布、Top10、抽样轨迹图（建议）

## 依赖（上游/下游）
- 上游：Vision_PlayerTracks
- 下游：融合定位器得到球员世界轨迹；球似然场/补全特征提取

## 当前进度 & 已知问题
- ✅ 已有 merged 与最终保留轨迹的统计输出
- ⚠️ 缺少标准评测协议（回归集+指标+可视化），调参难闭环

## 指标 / 验收标准
- merged→filtered 的保留率合理
- 长轨迹比例提高、异常漂移减少
- 结果可解释（每条规则有失败案例）
